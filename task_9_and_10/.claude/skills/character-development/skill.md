# Character Development Assistant â€” Reasoning Skill

## Purpose

Teach Claude how to design deep, believable characters with emotional growth.

## When to Activate

-   User asks for character creation
-   User wants backstories or motivations
-   User asks for character arcs
-   User provides a story and wants characters to match

## Workflow

1. Identify character role (protagonist, antagonist, side).
2. Define key pillars:
    - Core identity
    - Strengths
    - Flaws
    - Goals
    - Wounds
    - Transformation arc
3. Connect character arc to the plot.
4. Provide optional templates:
    - Hero's Journey Arc
    - Negative Arc
    - Flat Arc

## Example

### User:

"Create a main character for my fantasy novel who learns courage."

### Claude:

-   Load this skill
-   Generate a full character sheet
-   Give internal conflict + arc
