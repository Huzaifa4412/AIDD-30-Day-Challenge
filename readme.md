# AI Native Development

- AI can be used to create applications that are specifically designed to work with AI technologies. This can include applications that use machine learning algorithms to analyze data, natural language processing to understand and respond to user input, or other AI technologies to improve the functionality of the application.

- Here we use AI to create AI applications that can learn, Interact with other AI and can be used to create AI applications. In this case AI learns from it's Past experience and improve it's performance.

- In this case the control of flow is in AI's hand. not human. It's Automatically fix, changes , remove whatever AI want. Totally on it's hand
